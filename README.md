# ดาราจักร (Darachak) — Interactive 3D Astronomy Learning Platform

ห้องเรียนดาราศาสตร์ 3 มิติภาษาไทย ระดับพิพิธภัณฑ์วิทยาศาสตร์ — สร้างด้วย Three.js ล้วน ไม่ต้อง build ไม่ต้องโหลดไฟล์ texture (พื้นผิวดาวทุกดวงสร้างสดแบบ procedural)

**By Tee Jakkrit**

## ฟีเจอร์

### 🪐 โหมดระบบสุริยะ
- ดวงอาทิตย์เพลาสมา (custom GLSL shader) พร้อมโคโรนาและเปลวสุริยะ
- ดาวเคราะห์ครบ 8 ดวง + ดาวเคราะห์แคระ (ซีรีส, พลูโต) + ดาวหางฮัลเลย์พร้อมหางที่ชี้หนีดวงอาทิตย์เสมอ
- โลกมีเมฆเคลื่อนไหว ชั้นบรรยากาศเรืองแสง และดวงจันทร์โคจร · ดาวเสาร์มีวงแหวนหลายชั้นพร้อมช่องแคสสินี · ดาวพฤหัสบดีมีจุดแดงใหญ่
- แถบดาวเคราะห์น้อย 1,700 ก้อน · วงโคจรเรืองแสง · ป้ายชื่อภาษาไทยทุกดวง
- แตะดาวดวงไหน → กล้องบินซูมไปหา พร้อมแผงข้อมูล Hologram ภาษาไทย

### 🌌 โหมดท้องฟ้าจำลอง (Planetarium)
- หมู่ดาวสำคัญ 12 กลุ่ม จากพิกัด RA/Dec จริง พร้อมตำนานกรีกและชื่อไทยโบราณ (ดาวไถ ดาวจระเข้ ดาวลูกไก่)
- ดาวฤกษ์สำคัญ 9 ดวง + วัตถุท้องฟ้าลึก (เนบิวลา กระจุกดาว กาแล็กซี)
- **จำลองท้องฟ้าจริง** — เลือกจังหวัด วันที่ เวลา ระบบคำนวณ Local Sidereal Time แสดงตำแหน่งดาวตรงตามท้องฟ้าจริง ณ เวลานั้น

### 📚 ระบบการเรียนรู้
- 3 ระดับ: เด็ก / นักเรียน / ผู้เชี่ยวชาญ
- เสียงบรรยายภาษาไทย (Web Speech API)
- แบบทดสอบ 3 ระดับความยาก พร้อมคำอธิบายเฉลย
- รองรับมือถือเต็มรูปแบบ (ปัดหมุน 360°, pinch zoom, bottom-sheet)

## วิธีรัน

```bash
python3 -m http.server 5556
# เปิด http://localhost:5556
```

หรือ static server อะไรก็ได้ — เป็นไฟล์ HTML/CSS/JS ล้วน (Three.js โหลดจาก CDN)

### Deep links
- `?mode=sky` — เปิดโหมดท้องฟ้าจำลองทันที
- `?pick=saturn` — บินไปดูดาวเสาร์พร้อมข้อมูลทันที (ใช้ id: `sun`, `mercury`…`neptune`, `moon`, `pluto`, `comet` ฯลฯ)

## เทคโนโลยี

Three.js 0.160 · แผนที่ดาวจริงความละเอียด 2K · CSS2DRenderer สำหรับป้ายชื่อไทย · ไม่มี dependency อื่น

## เครดิตภาพพื้นผิวดาว

- แผนที่ดาวเคราะห์/ดวงจันทร์/ดวงอาทิตย์: [Solar System Scope Textures](https://www.solarsystemscope.com/textures/) (CC BY 4.0, อ้างอิงข้อมูลภาพถ่ายจาก NASA)
- Earth normal/specular maps: [three.js examples](https://threejs.org/) (MIT)
- ภาพวาดตำนานหมู่ดาว: [Stellarium](https://stellarium.org/) วาดโดย Johan Meuris (Free Art License)
- ภาพถ่ายวัตถุท้องฟ้าลึก: NASA/ESA/ESO ผ่าน Wikimedia Commons
- แคตตาล็อกดาว: Yale Bright Star Catalogue (BSC5)
- ดาวพลูโตและฉากหลังอวกาศ: procedural generation (seeded value-noise fBm)
