# ระบบสุริยะ — Interactive 3D Astronomy Learning Platform

ห้องเรียนดาราศาสตร์ 3 มิติภาษาไทย ระดับพิพิธภัณฑ์วิทยาศาสตร์ — Three.js ล้วน ไม่ต้อง build ตำแหน่งดาวทุกดวง**คำนวณจริงตามเวลา** (JPL ephemeris)

โดย **Tee Jakkrit**

## ฟีเจอร์

### 🪐 โหมดระบบสุริยะ

- พื้นผิวดาวทุกดวงเป็นภาพถ่ายจริง 2K (NASA imagery) — โลกมีทวีปจริง เมฆจริง และไฟเมืองฝั่งกลางคืน
- ตำแหน่งดาวเคราะห์ ดวงจันทร์ และดาวหางฮัลเลย์ ตรงกับท้องฟ้าจริง ณ เวลาปัจจุบัน (JPL Keplerian elements + สูตร Meeus)
- ความเร็วเวลา ×1 = เวลาจริง เร่งได้ถึง ×10 ล้าน พร้อมจอแสดงวันเวลาจำลอง
- ดวงจันทร์บริวารเด่น 5 ดวง (ไอโอ ยูโรปา แกนีมีด คัลลิสโต ไททัน) คลิกดูข้อมูลได้
- แตะดาว → กล้องบินไปจอดฝั่งสว่าง พร้อมหมุนชมรอบดาวอัตโนมัติ
- เครื่องคิดน้ำหนักตัวบนดาวแต่ละดวง + เฟสดวงจันทร์จริงคืนนี้ (ขึ้น/แรมกี่ค่ำ)

### 🌌 โหมดท้องฟ้าจำลอง (Planetarium)

- ดาวจริง 8,404 ดวงจาก Yale Bright Star Catalogue — ขนาดตาม magnitude สีตามอุณหภูมิดาว
- หมู่ดาว 19 กลุ่ม (จักรราศีครบ 12) พร้อมภาพวาดตำนานจริงจาก Stellarium
- ดวงอาทิตย์ ดวงจันทร์ ดาวเคราะห์ 5 ดวง ณ ตำแหน่งจริงตามจังหวัด/วัน/เวลาที่เลือก
- บรรยากาศสมจริง: ดาวตกสุ่ม ดาวกะพริบ แสงสนธยา เงาต้นไม้รอบขอบฟ้า ฟ้ากลางวันสีฟ้า
- ภาพถ่ายจริงจาก Hubble/ESO ของเนบิวลา กระจุกดาว และกาแล็กซีทุกชิ้น

### 🧪 เครื่องมือครูวิทยาศาสตร์

- โหมดสาธิต 5 ฉาก: เฟสดวงจันทร์ / ฤดูกาล / สุริยุปราคา / จันทรุปราคา / เทียบขนาดจริง — ทุกฉากมีจอเล็ก "มุมมองที่ตาเห็นจริง"
- ทัวร์นำชมอัตโนมัติ 10 จุด พร้อมเสียงบรรยายไทย
- [คู่มือครู](teacher.html) แผนการสอน 50 นาที 3 แผน ผูกตัวชี้วัด ว 3.1
- [เครื่องสร้างใบงาน](worksheet.html) พิมพ์ได้ พร้อมเฉลยแยกหน้า

### 📚 ระบบการเรียนรู้

- 3 ระดับ: เด็ก / นักเรียน / ผู้เชี่ยวชาญ — ปรับความลึกเนื้อหาและความยากแบบทดสอบ
- บทอ่านเจาะลึกภาษาไทย 34 เรื่อง ~100 หัวข้อ + เสียงบรรยายไทย (Web Speech API)
- แบบทดสอบ 3 ระดับพร้อมคำอธิบายเฉลย
- รองรับมือถือเต็มรูปแบบ (ปัดหมุน 360°, pinch zoom, bottom-sheet)

## วิธีรัน

```bash
python3 -m http.server 5556
# เปิด http://localhost:5556
```

หรือ static server อะไรก็ได้ — เป็นไฟล์ HTML/CSS/JS ล้วน (Three.js โหลดจาก CDN)

### Deep links

- `?mode=sky` — เปิดโหมดท้องฟ้าจำลองทันที
- `?pick=saturn` — บินไปดูดาวเสาร์พร้อมข้อมูล (ใช้ id: `sun`, `mercury`…`neptune`, `moon`, `io`, `titan`, `pluto`, `comet` ฯลฯ)
- `?demo=phases` — เปิดฉากสาธิต (`phases` / `seasons` / `solar-eclipse` / `lunar-eclipse` / `compare`)
